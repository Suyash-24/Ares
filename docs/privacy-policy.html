Privacy Policy

Last Updated: January 10, 2026

Your privacy is important to us. This policy explains what data Ares Bot collects, how it's used, and your rights.

1. Information We Collect

Ares Bot collects and stores limited data necessary for its functionality:

Server Data

  • Server ID: To identify and store server-specific configurations
  • Channel IDs: For features like welcome messages, logging, and starboard
  • Role IDs: For moderation, leveling rewards, and permission systems
  • Server settings: Custom prefix, enabled features, and configurations

User Data

  • User ID: To track XP/levels, warnings, and user-specific settings
  • Message counts: For leveling and statistics (not message content)
  • Voice time: Duration spent in voice channels for leveling
  • Moderation history: Warnings, mutes, and other moderation actions

✅ What We DON'T Collect:

  • Message content (except for logged mod actions)
  • Personal information (email, real name, etc.)
  • Private/DM messages
  • Voice conversations or recordings

2. How We Use Your Data

Data Type Purpose
Server ID & Settings Store and apply server-specific configurations
User ID & XP Data Track levels, leaderboards, and reward roles
Moderation Logs Keep record of warnings, bans, and mod actions
Statistics Message counts and voice time for server analytics

3. Data Storage & Security

All data is stored securely in our database with the following measures:

  • Data is stored on secure, access-controlled servers
  • Only essential data required for bot functionality is stored
  • Data is not sold, shared, or transferred to third parties
  • Regular security updates and monitoring are performed

4. Data Retention

We retain data as follows:

  • Server data: Retained while the bot is in your server. Deleted upon bot removal (may take up to 30 days)
  • User data: Retained while you are in servers using the bot. Can be deleted upon request
  • Moderation logs: Retained as long as the server configuration specifies

5. Your Rights

You have the following rights regarding your data:

Right to Access

You can request a copy of your data by contacting us.

Right to Deletion

You can request deletion of your data. Server owners can also reset all data using .reset commands.

Right to Opt-Out

Server owners can disable data-collecting features (leveling, statistics, etc.) at any time.

⚠️ To Request Data Deletion:

Join our support server and open a ticket, or contact the bot developers directly. Include your User ID or Server ID.

6. Third-Party Services

Ares Bot may use the following third-party services:

  • Discord API: Required for bot functionality
  • Lavalink: For music playback (no user data stored)
  • Database hosting: For secure data storage

7. Children's Privacy

Ares Bot is not intended for users under 13 years of age, in accordance with Discord's Terms of Service. We do not knowingly collect data from children under 13.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Significant changes will be announced in our support server. Continued use of the bot after changes constitutes acceptance.

9. Contact Us

For privacy concerns, data requests, or questions about this policy:

  • Join our support server
  • Contact the bot developers
